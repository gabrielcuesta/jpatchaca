Jpatchaca

Jpatchaca personal time tracker

Download this project as a .zip file Download this project as a tar.gz file

Jpathaca is an easy to use, free/open source time tracking apllication.

Get it NOW (requires java 6 installed)

JPatchaca will open as a tray icon: Middle-click, write your task name, press enter and that's it, left click to stop/start.

Jira integration

  • log your work on a issue directly from JPatchaca;
  • import issues automatically;

Own your data

All events are serialized to xml under /.jpatchaca, it means that even if the power goes down you don't lose your log.

Free as in freedom

Also, it's GPL3, feel free to get the code and change it however you want .
